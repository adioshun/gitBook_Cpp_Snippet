# Class 

`.`연산자로 객체의 멤버 접근 : 일반적 방법 
`->`연산자로 객체의 멤버 접근 : 객체 포인터 
 - `*` 와 `&` 이용활용  주소 정보 

## Class 선언 / 사용 

```cpp 
// 선언 
class Fish //대문자
{
 ....
}


//인스턴스 생성 

 Fish fish; //소문자 인스턴스명 
   
```

## 생성자와 소멸자 

클래스 내의 별도의 메소드 이다. 
- 생성자 : 클래스와 동일한 이름 
- 소멸자 : `~`클래스와 동일한 이름 


## 클래스의 상속 

부모 클래스 
```cpp
class NPC {
 ...
}
```

자식 클래스 
```cpp
class Grunt : public NPC{


}
```
